spojy.complete_profile_info module¶

‘ This module takes as input:
i> the username of the profile that you want to gather data about.
ii> network proxy (optional)
iii> the maximum number of asynchronous that can be sent at a time,this defaults to 77
and returns the following:
i> world rank of the user.
ii> total points of the user.
iii> a dictionary with languages used as the key and the number of problems solved with that language as the value. iv> a dictionary with problem tag as the key and the number of problems with that problem tag as value.

spojy.profile_info module¶

This module takes as input:
i>username of the user, whom you want to gather information about.
and returns:
i> the list of problem codes of the solved problems by the user.
ii> the list of problem codes of the unsolved problems by the user.
iii> world_rank of the user
i> total points of the user
